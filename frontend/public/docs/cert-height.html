🪜 הסמכת עבודה בגובה

Work at Height Certification Guide

🪜
🎯 מטרה ותחולה

מסמך זה מרכז את דרישות ההסמכה לעובד המבצע עבודה בגובה — כל עבודה שבה העובד עלול ליפול מגובה העולה על 2 מטר. ההסמכה הינה תנאי מחייב להעסקת עובד בעבודות סולמות, פיגומים, סלי הרמה, גגות, מבנים זמניים ופיגומים תלויים. ללא תעודת הסמכה בתוקף חל איסור מוחלט על העובד לבצע עבודה בגובה.

⚖️ בסיס חוקי ותקינה
  • תקנות הבטיחות בעבודה (עבודה בגובה), התשס"ז-2007.
  • פקודת הבטיחות בעבודה [נוסח חדש], התש"ל-1970.
  • תקנות הבטיחות בעבודה (ציוד מגן אישי), התשנ"ז-1997 — לעניין רתמות ובלמי נפילה.
  • ת"י 1849 — מערכות בלימת נפילה; ת"י 1139 — פיגומים.
  • הוראות מינהל הבטיחות והבריאות התעסוקתית בדבר מדריכים מוסמכים לעבודה בגובה.
📋 שלבי ביצוע מפורטים
  1. איתור מדריך עבודה בגובה המוסמך ע"י מינהל הבטיחות ומחזיק בתחום ההסמכה הרלוונטי (סולמות / פיגומים / סל הרמה / גגות).
  2. רישום העובד להדרכה עיונית בנושאי סיכוני נפילה, נקודות עיגון, חישוב מרחק נפילה וגורם נפילה.
  3. ביצוע הדרכה מעשית ייעודית לכל סוג עבודה (לדוגמה: עבודה על גגות שבירים נדרשת בנפרד מעבודה על פיגום).
  4. מבחן מעשי ועיוני; הצלחה מהווה תנאי לקבלת תעודת הסמכה.
  5. הנפקת תעודה אישית הנושאת את שם העובד, סוגי העבודה המורשים ותאריך התפוגה.
  6. תיוק העתק התעודה בתיק העובד ובתיק הבטיחות של הפרויקט.
  7. מעקב אחר מועד הרענון התקופתי וזימון העובד לרענון לפני פקיעת התוקף.
👥 תפקידים ואחריות
  • מעסיק — אחראי להעסיק רק עובדים מוסמכים ולממן את ההכשרה והרענון.
  • מדריך עבודה בגובה מוסמך — מעביר ההדרכה, מבצע המבחן ומנפיק התעודה.
  • ממונה בטיחות — מוודא תקפות ההסמכות ומנהל מעקב מועדי רענון.
  • העובד — אחראי להחזיק התעודה בתוקף ולהשתמש בציוד הנפילה בהתאם להדרכה.
🦺 ציוד ואמצעים נדרשים
  • רתמת גוף מלאה תקנית עם נקודות עיגון מאחור ובחזה.
  • בולם נפילה / חבל הצלה עם בולם אנרגיה תקני.
  • קסדת מגן עם רצועת סנטר; נעלי בטיחות נוגדות החלקה.
  • נקודות עיגון מאושרות בעלות עומס עבודה מתאים.
  • תעודת הסמכה אישית בתוקף נושאת העובד באתר.
