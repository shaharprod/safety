🩺 הסמכת מעניק עזרה ראשונה

First Aid Provider Certification Guide

🩺
🎯 מטרה ותחולה

מסמך זה מפרט את דרישות הסמכת מעניק העזרה הראשונה במקום העבודה. תקנות מקום העבודה מחייבות נוכחות מעניקי עזרה ראשונה מוסמכים בכל משמרת, ביחס למספר העובדים ולרמת הסיכון. ההסמכה ניתנת באמצעות קורס מוכר (מד"א או גוף מורשה) ומקנה כשירות להעניק טיפול ראשוני עד הגעת כוחות רפואה.

⚖️ בסיס חוקי ותקינה
  • תקנות הבטיחות בעבודה (עזרה ראשונה במקומות עבודה), התשמ"ח-1988.
  • פקודת הבטיחות בעבודה [נוסח חדש], התש"ל-1970.
  • הנחיות מגן דוד אדום לקורסי עזרה ראשונה והחייאה (CPR).
  • תקן תכולת ערכת עזרה ראשונה לפי מספר העובדים וסוג העבודה.
  • הוראות משרד העבודה בדבר יחס מעניקי עזרה ראשונה לעובדים.
📋 שלבי ביצוע מפורטים
  1. קביעת מספר מעניקי העזרה הראשונה הנדרשים לפי גודל כוח האדם ורמת הסיכון.
  2. רישום העובדים לקורס עזרה ראשונה מוכר (מד"א / גוף מורשה) בהיקף שעות מלא.
  3. הכשרה בנושאי החייאה, חבישות, עצירת דימומים, כוויות, שברים והלם.
  4. תרגול מעשי בבובת החייאה ובשימוש בדפיברילטור (AED) במידת הצורך.
  5. מבחן עיוני ומעשי והנפקת תעודת מעניק עזרה ראשונה לתוקף של 3 שנים.
  6. פרסום שמות מעניקי העזרה הראשונה ומיקום ערכות העזרה באתר.
  7. זימון לרענון לפני תום 3 השנים לחידוש התעודה.
👥 תפקידים ואחריות
  • מעסיק — מוודא נוכחות מעניק מוסמך בכל משמרת ומממן ההכשרה.
  • מעניק עזרה ראשונה — מעניק טיפול ראשוני ומוודא תקינות הערכה.
  • ממונה בטיחות — מנהל מעקב תוקף תעודות ומיקום ערכות.
  • אחראי משמרת — מוודא שמעניק מוסמך נוכח בפועל בכל זמן עבודה.
🦺 ציוד ואמצעים נדרשים
  • תעודת הסמכה אישית בתוקף (עד 3 שנים).
  • ערכת עזרה ראשונה מאוישת ומסומנת בהתאם לתקן.
  • דפיברילטור חצי-אוטומטי (AED) נגיש, היכן שנדרש.
  • שילוט מיקום ערכות ורשימת מעניקי עזרה ראשונה תורנים.
  • אמצעי מיגון אישי (כפפות, מסכת החייאה חד-פעמית).
