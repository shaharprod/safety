Project Safety File — Structure & Maintenance Guide

📁
🎯 מטרה ותחולה

תיק בטיחות הפרויקט הוא הקלסר המרכזי המרכז את כלל מסמכי הבטיחות הנדרשים לאתר העבודה לאורך כל חיי הפרויקט. מטרתו לרכז במקום אחד נגיש את ההוכחות לכך שמבצע הבנייה עומד בדרישות פקודת הבטיחות בעבודה ותקנות הבנייה, ולאפשר למפקח עבודה, למזמין ולממונה הבטיחות לאמת בכל רגע את מצב הציות. התיק מוחזק פיזית באתר ובעותק דיגיטלי מגובה.

📑 מבנה ותכולת המסמך

התיק מאורגן בחוצצים ממוספרים, כל חוצץ נושא רשימת תכולה (אינדקס) בתחילתו. סדר החוצצים:

  1. פרטי הפרויקט והמזמין, היתר בנייה והודעה לביטוח לאומי על פתיחת אתר.
  2. כתבי מינוי: מנהל עבודה, ממונה בטיחות, נאמני בטיחות.
  3. היתרים ורישיונות: עבודה בגובה, מנופים, חפירות, עבודה חמה.
  4. סקרי וניתוחי סיכונים (JSA) ותכנית ניהול הבטיחות.
  5. נהלי עבודה בטוחה ותכנית חירום.
  6. רישומי הדרכות, רשימות נוכחות והסמכות עובדים.
  7. בדיקות ציוד ותעודות כושר תקופתיות.
  8. דוחות בטיחות, אירועים, "כמעט-נפגע" וסיכומי סיורים.
📋 שלבי ביצוע מפורטים
  1. הקמת מבנה התיק בפתיחת האתר לפי האינדקס הסטנדרטי וקביעת מיקום קבוע באתר.
  2. הזנת המסמכים הראשוניים: היתר בנייה, מינויים, הודעת פתיחה.
  3. שילוב סקר הסיכונים ותכנית ניהול הבטיחות לאחר אישורם.
  4. עדכון שוטף: כל הדרכה, בדיקת ציוד או אירוע מתויקים תוך 48 שעות.
  5. סיור ביקורת חודשי לאימות שלמות התיק מול האינדקס.
  6. בסגירת האתר — ארכוב התיק המלא לתקופת השמירה החוקית.
👥 תפקידים ואחריות
  • מנהל העבודה — אחראי על קיום התיק באתר ועל עדכניות החוצצים התפעוליים.
  • ממונה הבטיחות — מבקר את שלמות התיק, מאשר נהלים וסקרים.
  • מבצע הבנייה — אחראי כולל על הקמת התיק וזמינותו למפקח.
  • מזמין העבודה — רשאי לדרוש עיון ולוודא קיום התיק.
🔄 מתי מעדכנים
  • עם כל שינוי בכוח אדם ניהולי (החלפת מנהל עבודה/ממונה).
  • בכל הדרכת בטיחות, בדיקת ציוד תקופתית או הוצאת היתר חדש.
  • לאחר כל אירוע בטיחות, "כמעט-נפגע" או ביקורת מפקח.
  • בכל שינוי מהותי בשלבי הביצוע או בהיקף העבודות.
